# virtual_tank
幻影坦克图片

## 使用说明
- 任何图片格式理论上都行
- 在文件夹按住shift右击打开控制台
- 格式：python virtual_tank.py 1.jpg 2.jpg（改名称一样的复制此段就行）
- 注意 1.jpg 是显示的缩略图 2.jpg是放大后的图
   并且前者是白底后者是黑底，自动调整也是对于颜色背景
   所以注意打字的话文字的颜色
